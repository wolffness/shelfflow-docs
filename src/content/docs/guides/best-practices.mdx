Ter prateleiras na homepage e so o comeco. O que realmente faz diferenca e **o que voce coloca nelas** e **como voce organiza**. Aqui estao as melhores praticas que aprendemos com lojistas reais.

## 1. Aplique a curva ABC

A regra de Pareto se aplica ao e-commerce: cerca de 20% dos seus produtos geram 80% da receita. Esses sao seus **produtos A** — eles devem ter maxima visibilidade.

**Como aplicar:**
- Identifique seus 10-20 produtos mais vendidos
- Crie uma prateleira "Mais Vendidos" com curadoria manual
- Coloque-a na posicao mais visivel (apos o banner)
- Coloque os mais vendidos **primeiro** no slider (posicao 1-3)

## 2. Use titulos que vendem

O titulo da prateleira e a primeira coisa que o visitante le. Bons titulos criam contexto e desejo.

| Generico | Estrategico |
|----------|-------------|
| Produtos | Mais Vendidos |
| Novidades | Acabou de Chegar |
| Categorias | Para Presentear |
| Destaques | Voce Vai Amar |

## 3. Crie prateleiras tematicas

Em vez de uma prateleira generica com produtos aleatorios, crie vitrines com **narrativa**:

- **"Look Completo"** — blusa + calca + acessorio
- **"Presentes ate R$ 50"** — faixa de preco acessivel
- **"Tendencia da Estacao"** — produtos sazonais
- **"Ultimas Unidades"** — urgencia para produtos com estoque baixo

## 4. Atualize regularmente

Uma homepage estatica passa a impressao de loja abandonada. Atualize suas prateleiras pelo menos **1x por semana**:

- Troque produtos que nao estao vendendo
- Adicione lancamentos
- Destaque promocoes vigentes
- Remova produtos sem estoque

<Aside type="tip">
  Use o **dashboard de analytics** para identificar quais produtos tem baixo CTR e substitua-los por novos candidatos.
</Aside>

## 5. Nao exagere no numero de prateleiras

Menos e mais. Uma homepage com 5+ prateleiras fica cansativa. Recomendamos:

- **1-2 prateleiras** para lojas pequenas (< 50 produtos)
- **2-3 prateleiras** para lojas medias (50-200 produtos)
- **3-4 prateleiras** para lojas grandes (200+ produtos)

## 6. Posicione estrategicamente

A ordem das secoes na homepage importa:

1. **Banner** (ja existe no tema)
2. **Mais Vendidos** — prateleira ShelfFlow, posicao "apos o banner"
3. **Destaques do tema** (ja existe)
4. **Lancamentos** — prateleira ShelfFlow, posicao "apos os destaques"
5. **Banners promocionais** (ja existe)
6. **Footer**

<Aside>
  Use o analytics para comparar o CTR entre posicoes. Se uma prateleira "apos o banner" tem CTR muito superior, considere mover as outras para posicoes mais altas tambem.
</Aside>
